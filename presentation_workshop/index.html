Qu'est-ce que PTL IR remote ?

Un μContrôleur esp8266

Emetteur IR (infra rouge)

Récepteur IR

Thermomètre et humidité

Led RGB

Luminosité (photorésistance)

Timer :
1'50'' min
Les composants à souder

les composants SMD sont déjà soudés

seuls les composants through hole sont à souder

Timer :
2'30'' min
Les composants à souder

deux headers

deux leds infra-rouge

une potoresistance

un recepteur infra-rouge

un thermomètre humidité senseur

Timer :
2'30'' min
Les soudures

bien chauffer la base de la patte

quand le pad est chaud, l'étain vient naturellement mouiller le pad

une bonne soudure est brillante et recouverte de flux (aspect gras)

Timer :
2'30'' min
Souder les leds infra-rouge (D1, D2)

plier les pattes à 90°, une courte et une plus plongue (attention à l'orientation)

attention à l'orientation la patte longue dans le trou rond, la courte dans le carré. Vérifier l'orientation du plat à la base de la led

utiliser le support pour les maintenir en place pendant la soudure

Timer :
3'15'' min
Souder le recepteurs infra-rouge (U1)

la boule est orientée vers l'exterieur de la carte

Timer :
3'15'' min
Le DHT22

placer le composant verticalement et le plier à 90°

bien aligner les trous de passage du DHT22 et de la carte

Timer :
3'15'' min
La photorésistance

placer la photorésistance et retourner la carte, elle doit être aproximativement à la même hauteur que le DHT22

l'orientation n'a aucune importance

Timer :
3'50'' min
Les headers de la carte et du µC

positionner les headers sur la carte et le µC

pour maintenir les headers en place souder une pine exterieur puis la pine opposée

souder le reste des pins

Timer :
3'50'' min
Vérifier que tout fonctionne bien

vérification visuelle

alimenter la carte...pas de magic smoke ?

le PTL-IR remote est en mode test, il devrait envoyer des données via MQTT

si le test passe, mise à jour du firmware via OTA

Timer :
6'00'' min
Seconnecter à la télécommande

si la led RGB clignotte en vert, PTL-IR fonctionne

Avec un smartphone se connecter au wifi LemanMake_PTL_workshop, password lemanmake

aller sur la page http://lmXX (XX numero de votre PTL-IR)

une page web devrait s'afficher...

Timer :
6'00'' min
Le serveur web de PTL-IR remote

page principale : température, humidité, luminosité

lmXX/led : contrôle de la led RGB

lmXX/ir : contrôle récepteur IR

lmXX/firmware : uploader un nouveau firmware

lmXX/ir_rest : envoyer un code IR via REST

lmXX/modes : les modes de la led RGB
